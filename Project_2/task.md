# Homework #2&#3: 非均匀介质中静电场的数值求解

## 背景

在磁共振成像（MRI）系统中，射频线圈在人体组织中产生电场。由于不同组织（如肌肉、脂肪、骨骼）具有不同的介电特性，电场分布需要求解变系数椭圆型方程。本作业考虑简化的二维模型。

## 1. 数学模型

**控制方程：**
$$-\nabla \cdot (k(x,y) \, \nabla u) = f(x,y), \quad (x,y) \in \Omega = (0,1) \times (0,1)$$

**扩散系数** （模拟组织介电特性的空间变化）：
$$k(x,y) = 1 + 0.9 \exp\left(-\frac{(x-0.5)^2 + (y-0.5)^2}{0.02}\right)$$

**源项** （模拟射频激励）：
$$f(x,y) = 8\pi^2 \sin(2\pi x) \sin(2\pi y)$$

**边界条件：**
$$u = 0 \quad \text{on} \quad \partial u$$

---

## 2. 作业要求

1. **离散化**：采用标准五点差分格式对方程进行离散。
2. **迭代求解**：实现以下三种迭代方法（取 $N=64$）：
    * Line Gauss-Seidel SOR
    * 共轭梯度法 (Conjugate Gradient Method)
    * 多重网格法 (Multigrid Method)
3. **参考解**：以 $N=256$ 的细网格数值解作为“精确解”。
4. **结果展示**：
    * 绘制三种方法的残差范数随迭代次数的收敛曲线（半对数坐标）。
    * 比较达到相对误差 $10^{-6}$ 所需的迭代次数。
    * 简要讨论各方法的收敛特性。
